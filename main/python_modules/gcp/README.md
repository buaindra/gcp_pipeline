## Pre-requisite:
### Installing Pip-
```shell
sudo apt-get install wget
wget https://bootstrap.pypa.io/get-pip.py
sudo python3 get-pip.py
```

### Installing/Activate Python Env-
```shell
sudo apt update
sudo apt install python3-venv

python3 -m venv env
source env/bin/activate
```
